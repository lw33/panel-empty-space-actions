# 顶部面板空白区动作

这是一个支持 GNOME Shell 46 到 50 的扩展，用来给顶部面板空白区域绑定鼠标动作。

## 功能

- 左键双击、中键单击、右键单击分别绑定不同动作
- 支持配置左键单击空白处时由扩展直接捕获，不触发面板默认行为
- 支持最小化、限时恢复、显示桌面、最大化/还原、关闭窗口、置顶/还原、隐藏所有普通窗口
- 每个鼠标手势都可以单独绑定自己的快捷键
- 支持调试日志
- 内置中文翻译

## 说明

- 只响应顶部面板空白区域
- 不会拦截按钮、菜单、时钟、托盘和指示器
- 在 X11 下，如果系统安装了 `xdotool`，自定义快捷键会优先按真实按键发送；如果未安装，会自动安全回退
- 在 Wayland 下，自定义快捷键会回退到已知 GNOME 动作和匹配到的自定义命令快捷键
- 扩展不会从自身设置中执行任意 shell 文本，只会触发 GNOME 内置动作，或触发用户已经在 GNOME 设置里配置好的自定义快捷键
