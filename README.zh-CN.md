# 顶部面板空白区动作

这是一个 GNOME Shell 46 扩展，用来给顶部面板空白区域绑定鼠标动作。

## 功能

- 左键双击、中键单击、右键单击分别绑定不同动作
- 支持最小化、限时恢复、显示桌面、最大化/还原、关闭窗口、置顶/还原、隐藏所有普通窗口
- 每个鼠标手势都可以单独绑定自己的快捷键
- 支持调试日志
- 内置中文翻译

## 说明

- 只响应顶部面板空白区域
- 不会拦截按钮、菜单、时钟、托盘和指示器
- 在 X11 下，如果系统安装了 `xdotool`，自定义快捷键会优先按真实按键发送
- 在 Wayland 下，自定义快捷键会回退到已知 GNOME 动作和匹配到的自定义命令快捷键

## 本地安装

```bash
./install.sh
```

## 调试日志

```bash
./debug.sh on
./debug.sh off
./debug.sh status
```
